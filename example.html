pr0 hat CSS-Auftrag


Obwohl ich die Arbeit von @retro sehr schätze, war pr0p0st für mich als HTML Harald nie wirklich eine Option, da die Flexibilität beim Erstellen eines Posts verloren geht. Also habe ich kurzerhand eine .css Datei namens pr0css.css erstellt, die alle wichtigen Farben für einen erfolgreichen pr0-Post mit HTML enthält.

Um das Stylesheet in HTML einzubinden, einfach im Header folgenden Eintrag machen: (Link in den Kommentaren)

   
      <link href="https://cdn.rawgit.com/Darvinx86/pr0css/master/pr0css.css" rel="stylesheet">

Natürlich kann man z.B. auch jQuery und Bootstrap inkludieren.


Die css besteht aus mehreren Klassen:

.richtigesGrau Färbt den Hintergund auf das von uns allen geliebte Grau.
.weiseSchrift
.orange
.edlerSpender
.neuSchwuchtel
.altSchwuchtel
.admin
.moderator
.fliesentischbesitzer
.gesperrt
.angenehmesGruen Angenehmes Grün
.megaEpischesBlau Mega Episches Blau
.olive Olivegrün des Friedens
.link Link, für z.B. @Gamb der Hurensohn
.cha0s Macht Text fett

HTML in PNG konvertieren

Jetzt müssen wir nur noch das von uns erzeugte .html Dokument in ein PNG umwandeln. Dafür eignet sich z.B. http://www.pdfconvertonline.com/htm-to-png-online.html. Einfach die .html Datei hochladen, den fertigen Post downloaden, auf Scheiße überprüfen und auf pr0 hochladen.
